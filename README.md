🌙 Terra do Nunca – Site inspirado em Peter Pan
📖 Sobre o projeto

Este projeto foi desenvolvido como parte de uma atividade escolar que consistia em escolher um livro da biblioteca e criar um site inspirado na história escolhida.

O livro selecionado foi Peter Pan, uma história clássica sobre um menino que nunca cresce e vive aventuras mágicas na Terra do Nunca junto com seus amigos.

O site apresenta elementos da história de forma interativa, permitindo que o usuário explore personagens, locais e acontecimentos importantes do livro.

🎯 Objetivo

O objetivo do projeto é transformar a narrativa do livro em uma experiência digital interativa, utilizando tecnologias de desenvolvimento web.

Além disso, o projeto busca:

apresentar a história de forma visual e divertida;

incentivar a leitura;

praticar conceitos básicos de HTML, CSS e JavaScript.

🧩 Funcionalidades do site

O site possui diferentes páginas que exploram elementos do livro:

Página inicial
Introduz o visitante ao universo da Terra do Nunca.

Personagens
Apresenta personagens importantes da história.

Mapa
Mostra os locais importantes da Terra do Nunca.

Quiz
Um pequeno quiz para testar o conhecimento sobre a história.

História
Uma linha do tempo interativa com capítulos da aventura de Peter Pan.

✨ Destaques do site

Algumas funcionalidades implementadas:

⭐ Fundo animado com estrelas

📜 Linha do tempo interativa da história

👆 Capítulos que abrem ao clicar

🎬 Animações ao rolar a página

🧚 Elementos visuais relacionados à Terra do Nunca

🛠 Tecnologias utilizadas

O projeto foi desenvolvido utilizando:

HTML – estrutura do site

CSS – estilo e animações

JavaScript – interatividade e efeitos
